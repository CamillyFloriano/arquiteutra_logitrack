# LogiTrack - Sistema de Otimização de Rotas e Entregas

Projeto desenvolvido para a disciplina de **Arquitetura de Software**.

O sistema LogiTrack tem como objetivo otimizar rotas de entrega, monitorar veículos em tempo real e melhorar a eficiência logística utilizando dados geolocalizados.
